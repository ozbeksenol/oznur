# Öznur Öğretmen Web Sitesi

Bu proje, Öznur Öğretmen için hazırlanmış modern ve responsive bir web sitesidir.

## Özellikler
- **Modern Tasarım:** Bootstrap 5 ve özel CSS ile hazırlanmış premium görünüm.
- **Responsive:** Mobil, tablet ve masaüstü uyumlu.
- **Animasyonlar:** AOS (Animate On Scroll) kütüphanesi ile yumuşak geçişler.
- **İçerik:** Sağlanan tüm metinler ve kategoriler entegre edildi.

## Nasıl Çalıştırılır?
Proje klasöründeki `index.html` dosyasını herhangi bir web tarayıcısında (Chrome, Safari, Firefox vb.) açmanız yeterlidir.

## Dosya Yapısı
- `index.html`: Ana sayfa
- `css/style.css`: Özel stil dosyası
- `js/main.js`: JavaScript efektleri
- `images/`: Görsel dosyaları
